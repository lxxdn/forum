Forum
---

## Introduction
This is a Rails4 project realizing a simple forum with MongoDB

## Requirements

* Ruby 2.0.0 +
* MongoDb 2.4.4 +

## Main techniques && GEMS

- Mongoid
- Devise
- Bootstrap
- CarrierWave
- Omniauth
- Simpleform

## Realised fonctions
1. creating a post and make comment  
2. Sign in & Sign up  
3. Upload picture as icon  
4. Authentification Google  
5. Catalog and topic
